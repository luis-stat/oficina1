# Projeto para a oficina de Streamlit:


## Funcionalidades

* **Filtros na Barra Lateral**: Permite filtrar a visualização dos dados por curso e por uma faixa específica de semestres.
* **Indicadores (KPIs)**: Exibe métricas em "tempo real", como total de alunos filtrados, média de horas de estudo e média do tempo de deslocamento.
* **Histograma Interativo**: Visualização da distribuição de frequências, permitindo ao usuário escolher qual variável analisar (estudo, deslocamento ou disposição).
* **Análise de Correlação**: Gráfico de dispersão que cruza o tempo de deslocamento com o nível de disposição.

## Como Executar o Projeto

Para rodar a aplicação localmente, siga os passos abaixo:

1. **Instale as dependências necessárias**:
   ```bash
   pip install streamlit pandas plotly statsmodels

2. ** Execute o comando do Streamlit:
   ```bash
   streamlit run app.py
